# Procesamiento-en-lote-de-datos-sobre-detenidos-colombianos-en-el-exterior.
Este repositorio contiene el desarrollo de una solución de procesamiento en lote orientada al análisis de datos relacionados con detenidos colombianos en el exterior utilizando Apache Spark para realizar un analisis exploratorio de daros (EDa)
El objetivo principal es aplicar técnicas de Big Data para la limpieza, transformación y análisis de datos, permitiendo identificar patrones relevantes asociados a:
* Países con mayor número de detenidos
* Tipos de delitos más frecuentes
* Distribución por género y grupo etario
* Situación jurídica de los detenidos
* Tendencias temporales de registros
  
El procesamiento se realiza utilizando DataFrames y RDDs de Spark, combinando operaciones de agregación, filtrado y transformación distribuida.
